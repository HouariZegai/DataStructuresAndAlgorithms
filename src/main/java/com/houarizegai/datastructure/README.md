## Data Structure

#### Simple array (fixed array size)
| Complexity | Value |
|:------:|:---------:|
| Access | O(1) |
| Search | O(n) |
| Insert | O(n) |
| Delete | O(n) |
| Space  | O(n) |

#### Dynamic array (Array list)
| Complexity | Value |
|:------:|:---------:|
| Access | O(1) |
| Search | O(n) |
| Insert | O(n) |
| Delete | O(n) |
| Space  | O(n) |

#### Linked List
| Complexity | Value |
|:------:|:---------:|
| Access | O(n) |
| Search | O(n) |
| Insert in First | O(1) |
| Delete in First | O(1) |
| Insert | O(n) |
| Delete | O(n) |
| Space  | O(n) |

Should i choose Linked List or Array List?
* A lot of add/delete => LinkedList
* Fixed element number => ArrayList
* Insert one time and a lot of access => ArrayList
