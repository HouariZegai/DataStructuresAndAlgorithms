## Arrays

#### Simple array (fixed size)
| Complexity | Value |
|:------:|:---------:|
| Access | O(1) |
| Search | O(n) |
| Insert | O(n) |
| Delete | O(n) |
| Space  | O(n) |

#### Dynamic array (flexible size)
| Complexity | Value |
|:------:|:---------:|
| Access | O(1) |
| Search | O(n) |
| Insert | O(n) |
| Delete | O(n) |
| Space  | O(n) |
